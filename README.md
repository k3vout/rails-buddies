# Rails Buddies

A web application that allows you to save your friends' information and access it from any part of the world thanks to the authentication feature made with the devise gem.

## Getting Started

### Installation

1. Clone the repo

   ```sh
   git@github.com:sevinchek/budget-app.git
   ```

### Usage

1. Install ruby gems dependencies

   ```sh
   bundle install
   ```

2. Install node dependencies

   ```sh
   npm install
   ```

3. Database creation

   ```
   rails db:setup
   ```

4. Run the web server

   ```sh
   rails server
   ```

5. Access to the website in a browser with the following link

   ```sh
   localhost:3000
   ```
